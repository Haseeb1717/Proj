Requirements

1. PHP version 8.1 or higher.

2. Apache server.

3. The following PHP extensions should be installed and enabled:

  • Ctype
  • cURL
  • DOM
  • Fileinfo
  • Filter
  • Iconv
  • Intl
  • Json
  • Hash
  • Mbstring
  • OpenSSL
  • PCRE
  • PDO
  • Session
  • Tokenizer
  • XML
  • mod_rewrite enabled.

These files and folders must be writable:

  • /public_html
  • + laravel/config/settings.php
  • + laravel/lang
  • + laravel/.env

This app uses Laravel 10, so check the documentation for more details.
